abapGit Repositories ADT Plug-In

This update site just contains abapGit Repositories plug-in and no further Eclipse content. Use it to install into SAP HANA Studio or any plain Eclipse instance with installed ADT Tools.

Installation
  • In Eclipse go to Help -> Install New Software...
  • In the install wizard enter URL http://eclipse.abapgit.org/updatesite/ and press Enter.
  • Select the desired package, e.g. ABAP Core Development for basic ABAP source code editing tools. Do not select features below the Uncategorized node.
  • Finish the wizard. A restart of the IDE is required.
