abapGit Repositories ADT Plug-In

This update site just contains abapGit Repositories plug-in and no further Eclipse content. Use it to install into ABAP Development Tools (ADT).

Installation
  1. In Eclipse, choose in the menu bar Help > Install New Software...
  2. Add the URL http://eclipse.abapgit.org/updatesite/
  3. Press Enter to display the available features.
  4. Select abapGit plugin for ABAP Development Tools (ADT) and choose Next.
  5. On the next wizard page, you get an overview of the features to be installed. Choose Next.
  6. Confirm the license agreementsb> and choose Finish to start the installation.
